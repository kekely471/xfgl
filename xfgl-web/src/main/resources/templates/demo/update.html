    自定义表单验证：该验证中将原有验证设计为可为空，若需非空验证请使用required。例：phone|required

    使用：引用自定义模块verifyE

    verifyE.verify();

    自定义验证列表：

    值 说明
    phone 手机号格式
    email 邮箱格式
    url 链接格式
    number 仅数字验证
    date 日期格式
    identity 身份证格式
    telephone 固定电话格式
    zipCode 邮编格式
    pass 密码验证6到12位，且不可有空格
    title 标题字符数验证：大于等于5

    回车键绑定事件通用方法：解决框架中绑定回车事件时出现的BUG

    基础使用：

    commons.bindEnterKey(callBack,closeMsg,time);

    参数：

    参数 类型 是否必填 描述
    callBack function 是 回车所绑定的回调函数，调用后会锁定回车键
    closeMsg boolean 否 当回调函数中有layer提示弹框时，true：回车键锁定，会在调用函数和关闭提示直接切换
    time number 否 回车键锁定后延时解锁

    方法：切换回车锁定状态

    commons.enterState(state);
